# BADING SI RONAN

### Running the Application

1. Install dependencies:

```
npm install
```

2. Start the application in development mode:

```
npm run dev
```

### Production Run PARA MABILIS UI

```
npm install
npm start
```

### If using Docker

```
docker compse up -d --build
```
